# react-redux-boilerplate

Subjectively built boilerplate for React + Redux apps. Created from tweaks and additions on top of [create-react-app](https://github.com/facebookincubator/create-react-app).

## Comes with
- [Basic ui components](https://github.com/renanpvaz/react-redux-boilerplate/tree/master/src/components)
- [Redux and Routing](https://github.com/renanpvaz/react-redux-boilerplate/blob/master/src/index.js)
- [.editorconfig](https://github.com/renanpvaz/react-redux-boilerplate/blob/master/.editorconfig)
- [Import aliases to top-level folders](https://github.com/renanpvaz/react-redux-boilerplate/blob/master/config/paths.js#L55)
- [Feather icons](https://feathericons.com/)
- [Redux saga](https://github.com/redux-saga/redux-saga)
